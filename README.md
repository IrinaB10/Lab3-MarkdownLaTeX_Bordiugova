# Лабораторная работа №3: Markdown и LaTeX

## Краткое описание

Данная лабораторная работа посвящена изучению языка разметки **Markdown** и основ **LaTeX** для оформления документации. В ходе работы были созданы примеры всех основных элементов Markdown, включая заголовки, списки, таблицы, цитаты, блоки кода, ссылки, изображения, чекбоксы, сноски, alert-блоки и математические формулы.

## Markdown-элементы

### Форматирование текста

- **Полужирный текст**
- *Курсив*
- ***Полужирный курсив***
- ~~Зачёркнутый текст~~
- `Моноширный текст (inline code)`

### Списки

#### Маркированный список
- Пункт 1
- Пункт 2
- Пункт 3

#### Нумерованный список
1. Первый пункт
2. Второй пункт
3. Третий пункт

#### Вложенный список
1. Git команды
   - `git init` — инициализация
   - `git add .` — добавление файлов
   - `git commit -m "message"` — создание коммита
   - `git push` — отправка на GitHub

### Цитата

> Markdown — это простой язык разметки, который позволяет форматировать текст без использования сложных редакторов. Он широко используется для документации в Open Source проектах.
