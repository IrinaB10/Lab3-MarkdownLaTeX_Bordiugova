# Списки в Markdown

## Маркированные списки

Создаются с помощью символов: *, -, +

- Пункт 1
- Пункт 2
- Пункт 3

* Альтернативный вариант 1
* Альтернативный вариант 2

+ Альтернативный вариант 3
+ Альтернативный вариант 4

## Нумерованные списки

Создаются с помощью цифр с точкой:

1. Первый пункт
2. Второй пункт
3. Третий пункт

## Вложенные списки

Используйте отступ (таб или 4 пробела):

1. Первый пункт верхнего уровня
   - Подпункт 1
   - Подпункт 2
   - Подпункт 3
2. Второй пункт верхнего уровня
   - Подпункт А
   - Подпункт Б

## Комбинированные списки

- Маркированный список
  1. Вложенный нумерованный
  2. Второй пункт
- Продолжение маркированного списка
  - Ещё один уровень вложенности
    - Третий уровень
