## Сноска

Markdown полезен в разработке[^1].

[^1]: Синтаксис Markdown поддерживается на GitHub, GitLab, в большинстве редакторов кода и систем документирования.

## Alert-блок

> [!TIP]
> Для экранирования специальных символов в Markdown используйте обратный слэш `\`. Например, чтобы написать *курсив* без форматирования, напишите `\*курсив\*`.

## Escape-символы

- \# Это не заголовок, а обычный текст с решёткой
- \*Это не курсив\*
- \_Это не подчёркивание\_
